### Key Features

#### 1. **Top Navigation Bar**
- **Logo**: Brand logo on the left (40x40px)
- **Search Bar**: Global search with autocomplete
  - Placeholder: "Search products, orders, customers..."
  - Keyboard shortcut: `Cmd/Ctrl + K`
  - Results grouped by type (Products, Orders, Customers)
- **Notifications**: 
  - Badge with unread count
  - Dropdown with recent notifications
  - Color-coded by type (success: green, warning: amber, error: red)
- **User Menu**:
  - Avatar with online status indicator
  - Dropdown: Profile, Settings, Help, Logout
  
#### 2. **Sidebar Navigation**
- **Width**: 240px (expanded), 64px (collapsed)
- **Collapsible**: Click hamburger to toggle
- **Module Groups**:
  - Overview (Dashboard, Analytics)
  - Operations (Inventory, Warehouse, POS)
  - Finance (Dashboard, Accounts, Reports)
  - Sales (CRM, B2B)
  - System (Employees, Locations, Settings)
- **Active State**: Highlighted with accent color + left border
- **Badges**: Show counts (e.g., "Inventory Â· 3 alerts")

#### 1. **Pipeline Board (Kanban)**
- **Columns**: Sales stages
- **Column Header**: Stage name + count + total value
- **Cards**: Deal cards (drag-and-drop)
- **Card Details**:
  - Company name
  - Deal value (large, bold)
  - Days until next action

  - Avatar of assigned rep
- **Features**:
  - Drag cards between stages
  - Click card â†’ Open detail modal
  - Filter by rep, value range, age
  - Sort by value, date, priority

- **Tabs**:
  - Info: Contact details, company info
  - Communications: Email/call history
  - Analytics: Engagement metrics, scoring
  - Activities: Timeline of interactions
- **Actions Bar**: Quick action buttons
- **Lead Score**: Visual indicator (0-100)
  - Color: Red (0-30), Amber (31-60), Green (61-100)
  - Fire icon if "hot" lead (>80)

#### 3. **Timeline**
- **Design**: Vertical timeline with icons
- **Items**: All customer interactions
- **Types**: Email, call, meeting, note
- **Features**:
  - Click to expand details
  - Add new activity inline

### Component Details

#### 1. **Channel Tabs**
- **Design**: Horizontal tabs with icons
- **Badge**: Unread count per channel
- **Active**: Highlighted with accent color

#### 2. **Conversation List (Left Panel)**
- **Items**:
  - Company/contact name
  - Preview text (truncated)
  - Timestamp (relative)
  - Unread indicator (blue dot)
  - Avatar/icon
- **Features**:
  - Click â†’ Open conversation
  - Search conversations
  - Filter by status (read/unread)
  - Sort by date, importance

#### 3. **Message Thread (Center Panel)**
- **Design**: Chat-like interface
- **Messages**:
  - Sender name + timestamp
  - Message bubble
  - Attachments (if any)
  - Read receipts
- **Features**:
  - Scroll to load older messages
  - Rich text formatting
  - Emoji picker
  - File attachment
  - Template insertion

#### 4. **Context Panel (Right Panel)**
- **Sections**:
  - Contact information
  - Related orders
  - Customer metrics
  - Scheduled messages
  - Quick actions
- **Features**:
  - Click order â†’ Navigate to details
  - One-click actions (email, SMS, call)
